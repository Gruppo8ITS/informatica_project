As mentioned before, our project focuses on enriching the ArCo Knowledge Graph by creating new semantic associations between chosen entities and properties. To do so, we proposed new RDF triples regarding the Musical Instrument Ontology. Hence, we suggested possible improvements especially with the regard to material and component properties, relating the following musical instruments:

PROMPTING TECHNIQUES

For what concerns the prompting techniques employed for our project we used four of them:

  1. Zero-shot technique → involves asking the model to perform a task without providing any examples. The model relies purely on its pre-trained knowledge to generate a response.
  2. Few-shot technique → provides the LLM with a few examples of the task before asking to perform it without any guideline.
  3. Chain of thought technique → encourages the model to think step-by-step improving its understanding and reasoning.
  4. Generated knowledge prompting → involves generating a knowledge background that can help the model perform a task more effectively.
